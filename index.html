My interest in coding led me to pursue computer science as a student. Initially, I enrolled in Santa Ana College to learn C++ with a focus on improving my IT skills and possibly pursuing a career in cybersecurity. However, an opportunity presented itself when a friend needed a website for his restaurant. I taught myself HTML, CSS, and Javascript to help him out, and this experience sparked my interest in game development. I created the classic game Tetris using C++ and the Raylib library. This project inspired me to create an AI that could play Tetris, which led me to explore machine learning. As I dived deeper into the world of data and machine learning, I became fascinated and now aspire to become a data scientist and eventually a machine learning engineer. Currently, I am learning Linear Algebra to prepare myself to learn Deep Learning.
